① 백업 생성

members → members_backup 복사 + JSON 파일 다운로드.
이미 백업이 있으면 덮어씁니다.

② 회원 데이터 초기화

members 컬렉션의 모든 문서를 삭제합니다.
백업 먼저 생성 후 진행하세요.

③ 롤백 (백업 → members 복원)

members_backup 컬렉션을 members로 복원합니다.
현재 members 데이터가 있으면 먼저 삭제됩니다.

④ 세션 · 팀편성 초기화

출결 관리의 이번 주 인원(weekly_session)과 팀편성(team_drafts)을 삭제합니다.
회원 데이터(members)는 건드리지 않습니다.

⚠️ 삭제된 데이터는 복구 불가 — 아래에 삭제 라고 입력 후 버튼을 눌러야 실행됩니다.

⑤ 관리자 권한 설정

이름으로 회원을 검색하여 역할(role)을 변경합니다.
